Get Started

Up and running in 4 steps.

No boilerplate decisions to make — just clone and build your product.

01

Clone & Install

Clone the repository and install dependencies.

git clone https://github.com/eng-kerolos-adel/kerodev-nextjs-rtl-starter.git my-project cd my-project && npm install
02

Setup Environment

Copy the example env file and add your Neon PostgreSQL connection string.

cp .env.example .env # Add DATABASE_URL="postgresql://..." to .env
03

Setup Database

Generate Prisma client and push the schema to your database.

npm run db:generate npm run db:push
04

Run Dev Server

Start the development server. You'll be redirected to /ar automatically — RTL is active.

npm run dev # → http://localhost:3000 → /ar
